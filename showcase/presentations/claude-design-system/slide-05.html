네 가지 규율

  • 슬라이드가 비어 보인다고 더미 통계나 장식 아이콘으로 채우지 않는다. 레이아웃과 스케일로 풀어라.
  • 브리프에 없는 섹션·페이지·카피를 추가하려면 먼저 사용자에게 물어라.
  • 데이터 슬롭 — 정보로 보이지만 의미 없는 숫자와 아이콘 — 을 금한다.
  • 잘라내는 것 자체가 디자인 도구다. 덜어냄을 두려워하지 마라.
