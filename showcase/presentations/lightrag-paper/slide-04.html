Architecture · Retrieval

핵심 ②Dual-Level 검색 패러다임

구체적 질문은 Low-Level, 추상적 질문은 High-Level — 두 가지 검색을 모두 활용

Low-Level

Specific · 정밀

특정 엔티티·속성·관계를 정밀하게 검색. 디테일 지향.

Example Query

"Who wrote 'Pride and Prejudice'?"

동작: Local keywords → 후보 엔티티 매칭

High-Level

Abstract · 통합

광범위한 주제·테마 통합. 다중 엔티티를 가로지르는 개념적 검색.

Example Query

"How does AI influence modern education?"

동작: Global keywords → 관계 + 다중 엔티티 통합

Retrieval Flow
i Keyword Extraction · local + global 분리
→
ii Vector Matching · 그래프 요소 매칭
→
iii Higher-Order · 1-hop 이웃 확장
